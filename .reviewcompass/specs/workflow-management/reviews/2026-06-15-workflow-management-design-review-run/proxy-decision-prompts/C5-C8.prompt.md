Source findings:

- claude-002: commit failure non-consumption is ambiguous, especially partial mutation.
- gpt-003: index may change after gate validation and before commit execution.
- gpt-004: consumption failure after successful commit is undefined.
- gemini-002: validation failure should not leave stale challenge / approval active.

Candidate options:

- A: Safe-side distinction. Validate immediately in the commit path against the exact index being committed. Security validation failure invalidates challenge / approval.
