# evidence/sessions/：セッション記録（整形済み発言全文転写・層 1）

会話転写（Claude / Codex の jsonl）を一次ソースとして機械抽出した、**整形・機微除去済みの
発言全文転写**を置く（PLC-DEC-007 追補・案 A）。裁定の遡及検証（決定台帳 ↔ 利用者発言の
突き合わせ）に用いる。生 jsonl の無加工取り込みはしない。

## 由来と方針

- 生成元：`tools/session-record-extractor.py`（単体）・`tools/session-record-backfill.py`（一括）。
- ツール結果は案 B（先頭 20 行＋末尾 20 行・上限つき）に縮約。内部思考は除外。ツール呼び出しは
  名前＋安全な引数に縮約（Bash のコマンド全文は出さない）。
- 機微除去 3 段（構造縮約・パターン除去・書き込み前スキャン）。書き込み前スキャンで秘密の
  残存が見つかったセッションは取り込まない（fail-closed）。除去ルールは
  `.reviewcompass/session-extraction/redaction-rules.yaml`（既定はコードに常駐）。
- ローカル転写には永続保証がないため、共有可能な正本としてここ（公開リポジトリ）に置く。
  取り込みは会話内容の公開を意味する（利用者了解のうえ・PLC-DEC-007）。

## 命名

`<YYYY-MM-DD>-<source>-<uuid>.md`
- `YYYY-MM-DD`：セッション最初の発言の日付（UTC）。
- `source`：`claude` または `codex`。
- `uuid`：セッションの完全な uuid（ファイル名由来。先頭数文字では時刻順 uuid が衝突するため完全形）。

人が読む層 2 の記録（日付・利用者指示・コミット一覧・触れたファイル）は `docs/sessions/auto-*.md`。
