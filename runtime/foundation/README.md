# runtime/foundation/

`foundation` 機能が所有する論理契約の正本を置く。コードではなくデータ優先の成果物として扱う（design.md §配置決定 1）。

## 配置されるファイル

- `layer1_framework.yaml`：レビュー段の論理契約（Step A/B/C/D）と役抽象名の正本（T-002）
- `metadata_contract.yaml`：実行メタデータ契約。必須項目一覧と語彙正本（T-003）
