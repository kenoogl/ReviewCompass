# 過去セッション議論記録のアーカイブ

本ディレクトリには、`Rwiki-v2-code-mod` リポジトリで作業していた時期に
利用者が手動で書き写していた議論記録を移管・保管する。

## 経緯

- ReviewCompass の再構築は最初 `Rwiki-v2-code-mod` リポジトリ内で計画立案され、
  そこでの議論を利用者が手動で 2 つのファイルに書き写していた
- セッション 18（2026-05-22）末で、§2.1「現リポジトリ＝素材として保全、
  新リポジトリ ReviewCompass＝再構築物として独立、両者を混ぜない」の原則に
  従い、ReviewCompass を作業ディレクトリに切り替えた
- 切り替え時点で過去の議論記録 2 件を本ディレクトリに移管。今後は
  `tools/session-log-converter.py` による自動記録（出力先：`docs/sessions/`）に
  切り替える

## 収録ファイル

- `ReviewCompass再設計.md`：セッション初期〜セッション 18 中盤までの議論記録。
  `Rwiki-v2-code-mod/docs/ReviewCompass再設計.md` からの複写
- `RC1.md`：セッション 18 中盤以降の議論記録。`/Users/Daily/Development/RC1.md`
  からの複写（git 管理外の独立ファイルだったため、移管時点で改めて保管）

## 注意事項

- 本ディレクトリ内の記録は手動書き写しのため、利用者の判断で抜粋・整形された
  内容になっている。Claude の応答そのままではない部分がある
- 今後の自動記録（`docs/sessions/`）は jsonl からの機械変換のため、内容は
  ほぼそのまま。両者を比べると粒度と網羅性が異なる
- 本ディレクトリの記録は **過去の経緯の参照用** として保管。新規追記は行わない
