# 項目2 整合性確保レビューの試行記録：proxy モードと書き込み後独立検証

最終更新：2026-05-31（セッション 42）

本メモは、計画書 §5.12 改訂の項目 2（残り 27 件の統合取り込み）を題材に、**§5.12.11 アサイン機構（proxy モード）** と **§5.8 補助層 D（書き込み後の独立検証、post-write-verification）** を初めて本格運用した試行の結果と所見を記録する。観察記録であり、方針正本（計画書・規律）ではない。

## 1. 何を試したか

- 対象：セッション 34 のアドホック変更 47 項目の整合性確保レビューの残務（残り 27 件）
- 事前作業：47 項目をセッション 34 の生ログ（idx 133）から全件復元し、事後検証で「解消済み 23 件」を除外。真にグレーな **4 件（項目 14・24・27・31）** を proxy 標準裁定にかけた
- 手段：実機 API（独立 3 系統＝ claude-sonnet-4-6／gpt-5.5／gemini-3.1-pro-preview）。起草者（メインセッション）は判定に入らず集約役・進行役に徹した（§5.12.11.7 起草者バイアス排除の鉄則）

## 2. proxy モードの効果（良かった点）

- **独立 3 系統の判定が、人（利用者）の最終判断の参考になった**。各モデルの理由・別案が判断材料として有用だった
- **効率よく集約・整理できた**。4 件を短時間で方針確定できた
- **外形指標（割れ・別案・僅差・質問返し）で発火を機械的に判定**し、収束項目（27）と割れ項目（14）を区別できた
- **別案で 3 系統が収束した項目 31** では、当初の案 1・案 2 になかった論点（計画書 3015 行の撤去済み規律名への参照切れ＝デッドリンクの修正）を 3 系統が独立に発見した。論文査読システムの「別案が採用方針になる」構造を再現した
- 起草者を判定から外す鉄則が機能し、係留バイアスの再混入を避けられた

## 3. 書き込み後独立検証（補助層 D）の効果

1 回目の検証で、起草者（メインセッション）の自己検証では見落とした**実質的な逸脱**を捕捉した：

- B：§5.9.3 に規律本体を書いてしまった（合意は「参照のみ」）
- D：§5.19.6 に撤去経緯の時系列叙述を書いてしまった（合意は「経緯は書かない」）
- A：§5.8 に根拠のない §5.12.11 への参照を挿入した
- E：運営ガイドに合意外の説明を付加した

いずれも起草者の自己点検では捕まらず、独立検証が捕まえた。**本検証の主目的（起草者が見落とす実質誤りの捕捉）を達成**した。

## 4. 課題：本質的でない指摘の収束（利用者指摘）

3 回の検証を通じて、次の挙動を観察した。

- 2 系統（Google・OpenAI）は 3 回目で **ALL_CLEAR**、1 系統（Anthropic／Sonnet）だけが**逐語・検証可能性レベルの指摘**を出し続けた
- 実質的な誤りは 1 回目で捕捉済み。2 回目以降の Sonnet の指摘は、具体例の有無・セッション番号の省略・用語「明示承認」の付加など、**内容の誤りではない逐語厳格なもの**だった
- 規律「1 件でも検出があれば深刻度問わず一律阻止」を厳格適用すると、1 系統が細かい指摘を出し続けて**収束しない恐れ**がある
- 今回は利用者判断「**2 系統 ALL_CLEAR で収束**」で対処した（これは post-write-verification の発効条件＝ OpenAI／Google の ALL_CLEAR と同じ基準）

## 5. 今後の検討（収束基準の論点）

「実質的な誤り」と「逐語・好みレベルの指摘」をどう切り分け、どこで収束とみなすかが、本規律の実運用上の課題として残る。候補：

- **多数決的収束**：独立 N 系統中 M 系統が ALL_CLEAR なら収束とみなす（例：3 系統中 2 系統）
- **深刻度フィルタ**：深刻度「高」のみ阻止し、「中・低」は記録に留める。ただし現規律は深刻度問わず一律阻止のため、改訂を要する
- **検証者への指示強化**：検証プロンプトで「内容の実質誤りのみ報告、文体・好み・逐語の一致は対象外」と明示する
- **繰り返し指摘の扱い**：同一系統が同一箇所を繰り返し指摘し、かつ他系統が問題視しない場合は収束とみなす

これらは post-write-verification 規律の改訂論点、または §5.12.11 の観察課題（dogfooding で確定）として継続検討する。本試行は「1 系統が逐語厳格で収束しにくい」挙動を実運用で初めて観察した事例として記録に残す。
