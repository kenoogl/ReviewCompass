1. **agreement_reflection**：合意内容（上記第 1 節）が書き込み（上記第 2 節）に正しく反映されているか
2. **reference_accuracy**：書き込み内の数値・出典・固有名詞（§番号、規律名、セッション番号等）が一致するか
3. **existing_record_consistency**：書き込みが既存正本（上記第 3 節の前後文脈）との齟齬を生まないか
4. **internal_logic**：書き込み内に論理矛盾がないか

```yaml

    criteria: agreement_reflection
```
